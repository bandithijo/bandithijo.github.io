Sudah lama saya ingin memiliki docking station untuk laptop ThinkPad X260 saya. Setelah melakukan riset dan mempertimbangkan berbagai opsi, akhirnya saya memutuskan untuk membeli ThinkPad USB 3.0 Pro Dock type 40A7.


## Konektivitas

Port yang tersedia pada ThinkPad USB 3.0 Pro Dock type 40A7:
1. 3x USB 3.0 â€“ 1 provides always-on mobile device charging
1. 2x USB 2.0
1. 1x USB 3.0 upstream connector
1. 1x DVI
1. 1x DisplayPort
1. 1x Gigabit Ethernet port
1. 1x Combination audio Port
1. 1x Security lock slot

Pro Dock type 40A7 ini menggunakan teknologi DisplayLink untuk menghubungkan monitor eksternal melalui USB 3.0. DisplayLink adalah teknologi yang memungkinkan transmisi video melalui koneksi USB dan Wi-Fi.

Dukungan DisplayLink di Linux telah meningkat pesat dalam beberapa tahun terakhir, dan sekarang banyak distribusi Linux yang mendukungnya dengan baik. Meskipun demikian, ada beberapa hal yang perlu diperhatikan saat menggunakan DisplayLink di Linux, seperti kebutuhan untuk menginstal driver tambahan dan kemungkinan adanya keterbatasan performa dibandingkan dengan koneksi video tradisional seperti HDMI atau DisplayPort.

Di Arch Linux, dukungan untuk perangkat DisplayLink dapat diaktifkan dengan menginstal paket `displaylink` (AUR). Setelah menginstal paket tersebut, saya dapat menghubungkan monitor eksternal ke docking station dan menggunakannya tanpa masalah. Hanya saja, performa grafis pada monitor eksternal mungkin tidak sebaik jika menggunakan koneksi langsung seperti HDMI atau DisplayPort.

## Pesan Penulis

Setelah menggunakan ThinkPad USB 3.0 Pro Dock type 40A7 selama beberapa minggu, saya tidak lagi menggunakan fitur DisplayLink karena performanya yang kurang memuaskan untuk kebutuhan saya. Saya lebih memilih untuk menggunakan koneksi langsung seperti HDMI atau DisplayPort untuk monitor eksternal saya. Dan menggunakan docking station ini hanya untuk menghubungkan perangkat USB lainnya seperti keyboard, mouse, dan hard drive eksternal.
