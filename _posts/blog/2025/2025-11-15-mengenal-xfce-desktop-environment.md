## Tentang XFCE

XFCE bukan dibaca "Eks Face", tapi dibaca "Ecks Eff See Eee".

Nama "Xfce" awalnya merupakan kepanjangan dari "XForms Common Environment". Namun, Xfce telah ditulis ulang (2x) dan sudah tidak lagi menggunakan toolkit XForms. Saat ini singkatan "Xfce" tidak memiliki arti apapun. Ada sebagian yang mengusulkan sebagai "X Freakin' Cool Environment". Hmmm ðŸ¤” boleh juga.

Meskipun, {{ page.description }}

XFCE adalah desktop yang akan masuk dalam list jika mencari desktop yang ringan.

Meskipun masuk dalam kategori desktop yang ringan, tapi XFCE memiliki tools utility pendukung yang lengkap layaknya desktop environment seperti GNOME dan KDE Plasma.

Tools utility ini sebenarnya merupakan project terpisah yang masing-masingnya merupakan bagian dari desktop environment XFCE.


### Core Components

1. **xfdesktop** (Desktop Manager) \
   *This program sets the background image and provides a root window menu, desktop icons or minimized icons and a windows list.*
1. **xfwm4** (Window Manager) \
   *It manages the placement of application windows on the screen, provides window decorations and manages workspaces or virtual desktops.*
1. **xfce4-panel** (Panel) \
   *The panel is a bar which allows you to have at all times program launchers, panel menus, a clock, a desktop switcher and more.*
1. **xfce4-session** (Session Manager) \
   *The session manager controls all aspects of the startup and the shutdown of the user session.*
1. **xfce4-settings** (Settings Manager) \
   *It allows you to configure every detail of your desktop environment.*
1. **thunar** (File Manager) \
   *Thunar is a modern file manager designed from the ground up to be fast and easy-to-use.*
1. **xfce4-appfinder** (Application Finder) \
   *Allows you to search, launch and find information about applications installed on your system.*
1. **xfce4-power-manager** (Power Manager) \
   *Manages the power sources of the computer and connected devices.*
1. **libxfce4ui** (Xfce Libraries) \
   *Widgets library for the Xfce desktop environment.*
1. **xfconf** (Xfconf) \
   *Simple client-server configuration storage and query system.*


### Applications

1. **xfce4-terminal** (Terminal Emulator) \
   *Terminal is a modern terminal emulator featuring tabs and transparent backgrounds.*
1. **Xfburn** \
   *Xfburn is an application to create and burn CD's and DVD's.*
1. **ristretto** (Image Viewer) \
   *Ristretto is a lightweight image viewer.*
1. **ristretto** (Image Viewer) \
   *Ristretto is a lightweight image viewer.*
1. **xfce4-screensaver** (Screen Saver) \
   *A simple screen saver and locker.*
1. **xfce4-screenshooter** (Screen Shooter) \
   *An application to take screenshots.*
1. **mouosepad** (Text Editor) \
   *A simple text editor.*
1. **xfce4-notifyd** (Notifications) \
   *Easily themable notification daemon with transparency effects.*
